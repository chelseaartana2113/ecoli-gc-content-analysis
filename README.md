# GC Content analysis of E.Coli genome

this project analyzed GC content across E.coli genome using a sliding window approach

## method
window size=5000
step size=500

## results
Average GC content: 50,16%
The lowest GC content: 32,1%
The highest GC content: 62,7%

three plots were generated:
- GC Content across genome E.coli
